这是我的第一个博客项目，用 [Astro](https://astro.build) 搭建，缘起于一个周末的好奇心。

代码写得不算漂亮，架构也说不上优雅。每一个功能都是边查边写、边写边改——有时候改着改着还会把自己绕进去，只好 Ctrl+Z 回到原点，重新来过。

但这件事本身，就很有意思。

如果你碰巧路过这里，看到了这些代码，欢迎提 issue，欢迎指正，甚至欢迎笑话我写得丑——毕竟，这正是我把它放在这里的原因。

> **成长的第一步，是承认自己什么都不会。**

---

## 技术

说实话，没什么好炫耀的：

- **Astro 5** — 刚学会，只会用最简单的静态生成
- **Tailwind CSS 4** — 样式全靠它，不然页面更丑
- **Vercel** — 一键部署，省去了我折腾服务器的功夫
- **Giscus** — 评论区，感谢 GitHub 慷慨的免费额度

## 功能

- [x] 首页 + 文章列表 + 文章详情 + 关于页
- [x] 暗色主题（因为我不会配亮色）
- [x] 标签筛选
- [x] RSS & Sitemap
- [x] 评论
- [ ] …还有很多想做的，正在慢慢学

## 如何运行

```bash
npm install
npm run dev
```

然后在浏览器打开 `http://localhost:4321`。
